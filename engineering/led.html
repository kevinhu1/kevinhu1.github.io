In this competition, we were given our challenge the night before and were required to come up with a design and look over all datasheets overnight. The challenge was this: implement a programmable circuit which uses a Piezo Buzzer. The following day, we were given six hours to wire, program, and debug our circuit to be presented the evening of.

Objective of the Game

A row of LEDs turn on, one at a time from bottom to top, then turns off from top to bottom (ie. filling up meter) at a given speed. The player must press the button at the moment when all LEDs are on. If the user succeeds, the game moves on to the next round where the speed increases, for up to five rounds. If the user misses (the LEDs can run infinitely until an input is read), the game will reset to round one. If the user wins all five rounds, the LEDs will flash simultaneously five times, and the game will reset. The Piezo Buzzer will beep for the duration of when all LEDs are on, both during each round and at the end of the game.
