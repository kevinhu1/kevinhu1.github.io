Course Objectives

Measurement of TTL Characteristics. Measuring current and voltages on inputs and outputs of digital circuits.
Micro-Controllers. Understanding their use in control systems and how to program them using C and MPLab. Implementing a PIC micro-controller in several assignments including a traffic light system and a button input counter.
Computer Reports. Researching a computer processor to understand the overall computer system and how the process works in tandem with its system, with emphasis on writing style in a technical format.
Verilog HDL and FPGAs. Understanding that Verilog is a hardware description language, ie. teaching its similarities and differences from C (wires and registers, combinational vs sequential, parallel, blocking vs nonblocking, etc). Using this understanding of Verilog to implement a simplified pipeline processor using a simplified MIPS onto an FPGA.
